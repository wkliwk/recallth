Layout Techniques Index
01 — Diagonal Split
Background splits diagonally, creating two visual zones. Content anchored to the left with large stacked typography. CTA in the lower zone. Creates dramatic first impression.
02 — Card Stacking
Apple Wallet-style depth. Completed cards pull forward with more detail. Locked cards recede into the background. Front card gets shadow + border treatment. Creates progression feeling.
03 — Broken Grid
Cards overlap each other, break alignment. Profile % overlaps with alert card. AI card rotated -1deg. Activity card at +0.5deg. Creates dynamic, editorial energy.
04 — Editorial Asymmetric
Oversized 44px title. Alternating narrow/wide column pairs (100px icon + fluid detail). Each item has different column ratio. Magazine editorial feel.
05 — Full-Bleed Immersive
AI responses break out of normal padding (negative margins). Purple gradient band spans full width. User bubbles stay contained. Creates hierarchy between user and AI.
06 — Connected Timeline
Vertical gradient line (red→amber→green) connects severity nodes. Each node has a dot indicator on the line. Cards branch off the timeline. Tells a visual severity story.
