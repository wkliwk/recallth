What makes v4 different
Typography
Instrument Serif (italic) for headings — editorial, luxury feel. Mixed with Inter for body. Large italic numbers as design elements.
Aurora Blobs
Organic morphing shapes (morph-blob animation) with blur create flowing aurora backgrounds. Each screen has unique placement.
Gradient Borders
Animated gradient borders (mint → violet → rose) on hero cards create depth and draw attention without being loud.
Color Palette
Mint + Lavender + Rose — softer, more nuanced than v2's green+purple. Each color has a semantic role (mint=health, violet=AI, rose=alerts).
Spatial Depth
Multiple glass layers (glass-1, glass-2) + floating blobs create parallax-like depth. Cards feel like they exist in 3D space.
Rounded 48px
Larger phone border-radius (48px vs 44px), 20px card radius, pill-shaped elements. Everything feels softer and more organic.
