Layout Structure
Shell: 100vw × 100vh, flex row. Fixed sidebar + flex-1 main area.
Sidebar: 240px fixed width, white bg, no scroll. Flex column with logo top, nav middle, user bottom.
Top bar: 60px fixed height, white bg, border-bottom. Contains page title + search + actions.
Content area: flex-1, overflow-y: auto, 32px padding all sides. Max practical content width ~880px inside the main panel.
Two-column grid: grid-template-columns: 1fr 400px, 24px gap. Left for schedule, right for AI/alerts/recent.
Sidebar Detail
Logo area: 28px top padding, logo icon (32×32, 10px radius, #E07B4A bg) + wordmark "Recallth" in DM Serif Display 22px.
Nav items: 40px tall, 10px border-radius, 10px vertical padding, 14px font. Active = orange tint bg + orange text. Icons are 20×20 stroke SVGs.
Active indicator: background rgba(224,123,74,0.1), no left border stripe — relies on bg color only.
User section: border-top, flex row, 36×36 avatar (orange, 10px radius), name in 600 weight, plan in ink3.
No collapse on 1280px — sidebar stays fixed. Below 1024px (tablet) it becomes a hamburger menu (out of scope for this wireframe).
Greeting & Stats Strip
Greeting: Sub-label "Good morning" in ink2 14px, name in DM Serif Display 32px ink1. Date badge right-aligned, white card style.
Stats strip: 4-column equal grid, white cards, 1px border. Value in DM Serif Display 28px (colored per type). Label in ink3 12px uppercase.
Stats content: 5 = supplement count (ink), 0 = conflicts (green), 14d = streak (orange), AI = powered badge (purple).
Zero conflicts: Stat card stays green — not amber. Only shows amber if >0 conflicts detected.
Today's Schedule Panel
Three time blocks: Morning (amber tint), Pre-Workout (blue tint), Evening (purple tint). Each block is a rounded card 14px radius.
Block header: Icon + label + time-badge pill. Time badge is white/border, floated right via margin-left: auto.
Pill cards: White, 1px border, 10px radius. Color dot (8px circle) + name (500 weight, 13px) + dose line (ink3, 11px). Flex-wrap so they reflow.
Empty state: If no supplements in a time slot, show dashed-border "Add supplement" card instead of pill-grid.
Interaction: Clicking a supplement card opens a details drawer or modal. Not part of this dashboard wireframe.
Right Column: AI Ask Card
Card bg: #F5F3FF (purple-bg), border #DDD6FE. 20px padding, 14px radius.
AI icon: 36×36, 10px radius, solid purple #7C3AED, white sparkle glyph. Matches mobile Chat FAB style.
Input row: White bg, border #DDD6FE, 12px radius. Placeholder text + send button (purple square, 28×28) on the right edge.
Quick prompts: 3 pre-set chips. White bg, #DDD6FE border, 8px radius. Full-width stacked. Clicking a chip pre-fills the input and focuses it.
Routing: Submitting the input navigates to /chat with the message pre-loaded. Does NOT open a modal on the dashboard.
Right Column: Conflict Alert
Visibility: Only shown if >0 conflicts exist. If 0 conflicts, this entire card is hidden (not replaced with empty state).
Card bg: #FFFBEB (amber-bg), border #FEF3C7. Icon is 28×28 amber-tinted square with ⚠ glyph.
Copy pattern: "[Drug A] + [Drug B] — [severity] interaction. [One-line consequence]." Max 2 conflicts shown in this card; "View all N conflicts →" if more.
CTA: "View full report →" in orange, links to /cabinet with conflict filter active.
Right Column: Recent Conversations
Card: White bg, border, 14px radius, no padding (rows go edge-to-edge with their own padding).
Row icon: 32×32, 9px radius. Purple-tinted for AI chats, green-tinted for cabinet events (pill was added/removed).
Row content: Title 13px 500 + meta line 11px ink3. Arrow › right-aligned. Entire row is clickable, navigates to /chat/:id.
Limit: Show 3 most recent. "View all conversations →" link below the card.
Top Bar Detail
Search bar: Max-width 420px, centered between title and actions. On focus, border turns orange. On submit, navigates to /chat with query.
Notification bell: 36×36 white card button. Amber dot indicator when unread. Clicking opens a dropdown (not in this wireframe).
Avatar: 36×36, same orange+initial style as sidebar. Clicking opens a small dropdown with Settings / Sign Out.
Tokens Used
Colors: orange #E07B4A · ink1 #2A221A · ink2 #7A6A5A · ink3 #B0A898 · sand #F2EDE4 · page-bg #FBF9F5 · border #EDE8E0
Supporting: amber #D97706 · green #059669 · purple #7C3AED (from mobile palette — add to design system)
Fonts: DM Serif Display (headings, stats, wordmark) · DM Sans 400/500/600/700 (all UI text)
Radii: 10px (inputs, small cards) · 12px (stat cards) · 14px (main cards) · 16px (shell)
Spacing base: 4px grid. Primary gutters: 24px content gap, 32px content padding, 12px card padding.
